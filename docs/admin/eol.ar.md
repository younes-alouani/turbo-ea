# إدارة نهاية العمر (EOL)

تساعدك صفحة إدارة **EOL** (**Admin > Settings > EOL**) على تتبّع دورات حياة المنتجات التقنية عبر ربط بطاقاتك بقاعدة البيانات العامة [endoflife.date](https://endoflife.date/).

![إدارة EOL](../assets/img/en/29_admin_settings_eol.png)

## لماذا تتبّع EOL؟

معرفة متى تصل المنتجات التقنية إلى نهاية العمر أو نهاية الدعم أمر بالغ الأهمية من أجل:

- **إدارة المخاطر** — البرمجيات غير المدعومة عبء أمني
- **تخطيط الميزانية** — خطّط للترحيلات والترقيات قبل انتهاء الدعم
- **الامتثال** — تتطلّب لوائح كثيرة برمجيات مدعومة

## البحث الجماعي

تفحص ميزة البحث الجماعي بطاقات **Application** و**IT Component** لديك وتعثر تلقائيًا على المنتجات المطابقة في قاعدة بيانات endoflife.date.

### تشغيل بحث جماعي

1. انتقل إلى **Admin > Settings > EOL**
2. اختر نوع البطاقة المراد فحصه (Application أو IT Component)
3. انقر **Search**
4. يُجري النظام **مطابقة ضبابية** مقابل كتالوج منتجات endoflife.date

### مراجعة النتائج

لكل بطاقة، يُرجِع البحث:

- **درجة المطابقة** (0–100%) — مدى تطابق اسم البطاقة مع منتج معروف
- **اسم المنتج** — منتج endoflife.date المطابق
- **الإصدارات/الدورات المتاحة** — إصدارات إطلاق المنتج مع تواريخ دعمها

### تصفية النتائج

استخدم عناصر التحكّم في التصفية للتركيز على:

- **All items** — كل بطاقة جرى فحصها
- **Unlinked only** — البطاقات غير المرتبطة بعد بمنتج EOL
- **Already linked** — البطاقات التي لها بالفعل رابط EOL

يعرض ملخّص إحصائي: إجمالي البطاقات المفحوصة، والمرتبطة بالفعل، وغير المرتبطة، والمطابقات المعثور عليها.

### ربط البطاقات بالمنتجات

1. راجع المطابقة المقترحة لكل بطاقة
2. اختر **إصدار/دورة المنتج** الصحيح من القائمة المنسدلة
3. انقر **Link** لحفظ الارتباط

بمجرد الربط، تعرض صفحة تفاصيل البطاقة **قسم EOL** يتضمّن:

- **اسم المنتج وإصداره**
- **حالة الدعم** — مرمّزة بالألوان: مدعوم (أخضر)، يقترب من EOL (برتقالي)، نهاية العمر (أحمر)
- **التواريخ الرئيسية** — تاريخ الإطلاق، نهاية الدعم النشط، نهاية الدعم الأمني، تاريخ EOL

## تقرير EOL

تُغذّي بيانات EOL المرتبطة [تقرير EOL](../guide/reports.md)، الذي يوفّر عرض لوحة معلومات لحالة دعم مشهدك التقني عبر جميع البطاقات المرتبطة.
