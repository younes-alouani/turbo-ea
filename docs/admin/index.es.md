# Administración

![Menú de Usuario con Opciones de Administración](../assets/img/es/19_menu_usuario.png)

La sección **Admin** está disponible para usuarios con el rol **Admin**. Proporciona herramientas para configurar el modelo de datos, el acceso de usuarios, las integraciones y la configuración general del sistema.

| Página | Descripción |
|--------|-------------|
| [Configuración General](settings.md) | Apariencia (logo, favicon, moneda), idiomas habilitados, año fiscal, correo electrónico (SMTP) e interruptores de módulos (BPM, PPM, GRC) |
| [Metamodelo](metamodel.md) | Tipos de ficha, campos, secciones, subtipos, tipos de relación, cálculos, etiquetas, principios EA y el grafo del metamodelo |
| [Cálculos](calculations.md) | Fórmulas definidas por el administrador que calculan automáticamente valores de campos al guardar fichas |
| [Etiquetas](tags.md) | Grupos de etiquetas y etiquetas para la categorización flexible de fichas |
| [Usuarios y Roles](users.md) | Gestión de usuarios, roles a nivel de aplicación y configuración de permisos |
| [Autenticación y SSO](sso.md) | Configuración de inicio de sesión único (Microsoft Entra ID, Google, Okta, OIDC genérico) y flujos de invitación |
| [Capacidades de IA](ai.md) | Sugerencias de descripción por IA con LLMs (Ollama, proveedores compatibles con OpenAI) y contexto de búsqueda web |
| [Integración MCP](mcp.md) | Servidor del Protocolo de Contexto de Modelos para acceso de herramientas de IA (Claude Desktop, Copilot, Cursor) |
| [Referencia de la API](api.md) | Especificación OpenAPI 3 en vivo embebida en el manual de usuario (sin necesidad de un backend en ejecución), además de autenticación, permisos y consejos para la generación de código |
| [Fin de Vida (EOL)](eol.md) | Vinculación masiva de datos EOL desde endoflife.date a fichas de tecnología |
| [Encuestas](surveys.md) | Encuestas de mantenimiento de datos dirigidas a tipos de fichas y partes interesadas específicas |
| [Portales Web](web-portals.md) | Portales públicos de solo lectura accesibles sin autenticación |
| [Integración ServiceNow](servicenow.md) | Sincronización bidireccional entre fichas de Turbo EA y CMDB de ServiceNow |
