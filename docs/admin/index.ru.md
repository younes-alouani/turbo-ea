# Администрирование

![Меню пользователя с параметрами администрирования](../assets/img/ru/19_menu_polzovatelya.png)

Раздел **Администрирование** доступен пользователям с ролью **Администратор**. Он предоставляет инструменты для настройки модели данных, управления доступом, интеграций и системных параметров.

| Страница | Описание |
|----------|----------|
| [Общие настройки](settings.md) | Внешний вид (логотип, фавикон, валюта), доступные языки, финансовый год, электронная почта (SMTP) и переключатели модулей (BPM, PPM, GRC) |
| [Метамодель](metamodel.md) | Типы карточек, поля, секции, подтипы, типы связей, вычисления, теги, принципы EA и граф метамодели |
| [Вычисления](calculations.md) | Формулы, определённые администратором, автоматически вычисляющие значения полей при сохранении карточек |
| [Теги](tags.md) | Группы тегов и теги для гибкой классификации карточек |
| [Пользователи и роли](users.md) | Управление пользователями, роли на уровне приложения и настройка разрешений |
| [Аутентификация и SSO](sso.md) | Настройка единого входа (Microsoft Entra ID, Google, Okta, универсальный OIDC) и процесс приглашений |
| [Возможности ИИ](ai.md) | Предложения описаний от ИИ на базе LLM (Ollama, провайдеры, совместимые с OpenAI) и контекст веб-поиска |
| [Интеграция MCP](mcp.md) | Сервер Model Context Protocol для доступа инструментов ИИ (Claude Desktop, Copilot, Cursor) |
| [Справочник API](api.md) | Live-спецификация OpenAPI 3, встроенная в руководство пользователя (запущенный бэкенд не нужен), плюс аутентификация, права и советы по генерации кода |
| [Конец жизненного цикла (EOL)](eol.md) | Массовая привязка данных EOL из endoflife.date к технологическим карточкам |
| [Опросы](surveys.md) | Опросы для поддержания данных, направленные на определённые типы карточек и заинтересованных лиц |
| [Веб-порталы](web-portals.md) | Публичные порталы только для чтения, доступные без аутентификации |
| [Интеграция ServiceNow](servicenow.md) | Двусторонняя синхронизация между карточками Turbo EA и CMDB ServiceNow |
